# stress-performance-matb-analysis
A computational analysis of multimodal physiological signals (EDA, BVP, HRV) to predict operator performance degradation during cognitive overload using the NASA MATB-II simulation environment.
This project utilizes the Neurophysiological Dataset of Stress Resilience During Human-Computer Interaction (Roy & Nuamah, 2026). The raw data (PPG and MATB-II logs) can be downloaded from PhysioNet.
